Detailed Description

Main sketch — FNIRSI DPS-150 Power Supply Control.

Entry point for the Processing application. Sets up the window, initialises the GUI and protocol layer, and routes Processing callbacks (draw, serial, mouse, keyboard) to the appropriate handlers.

Window size: 1100 x 720 pixels.
