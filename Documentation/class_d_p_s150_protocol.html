Detailed Description

Encapsulates all DPS-150 serial protocol state and methods.

Handles connection management, packet send/receive, register parsing, polling, data-logging, and graph-history buffering. Public fields expose the latest readings for the GUI to display; public methods provide write-back commands for controlling the PSU.

Usage

psu.connectToPort("/dev/cu.usbmodem...");
psu.sendSetVoltage(5.0);
psu.sendOutputOn();
