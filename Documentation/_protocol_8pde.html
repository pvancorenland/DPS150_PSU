Detailed Description

Serial protocol layer for the FNIRSI DPS-150 power supply.

Defines the binary packet format, register map, and the DPS150Protocol class that encapsulates all communication state and methods. A single global instance psu is created for use by all other tabs.

Packet format (request): F1 <cmd> <reg> <len> [data...] <chk> Packet format (response): F0 <cmd> <reg> <len> [data...] <chk>

Checksum = sum of bytes from reg through last data byte, AND 0xFF.
